Crosstalk Bio

About

Crosstalk Bio is a community building a decentralized hardware layer for conducting experiments. Each Crosstalk Bio project features an Internet of Things (IoT) device, enabling scientists from various locations to participate openly in experiments simultaneously. We believe this approach will foster learning diversity and improve reproducibility, thereby accelerating the rate of biomedical discoveries. To learn more, read our whitepaper draft here or join the discussion on Discord.
